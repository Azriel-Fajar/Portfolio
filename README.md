Personal portfolio site for Azriel, founder of Rielcode. Showcases his identity, skills, services, and past client work.

**Status:** In Progress
**Type:** Portfolio site
**Stack:** PHP, Bootstrap 5.3.8, vanilla JS, AOS, ScrollReveal
**Started:** 2026-04-27
